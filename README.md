# 🌃 星陨之夜 · NEON NIGHTFALL

一款赛博朋克风格的社交推理（类狼人杀）网页游戏。三大阵营、十余种角色，在霓虹夜色中斗智斗勇！

## 🎮 玩法

1. 在设置界面选择游戏人数（6/8/10/12/14 人局）和你的身份
2. 游戏按 **夜晚 → 白天 → 投票** 循环进行
3. 夜晚使用技能，白天发言讨论，投票放逐嫌疑人
4. 率先消灭对方阵营即可获胜

## ⚔️ 三大阵营

| 阵营 | 胜利条件 |
|------|----------|
| 天星 | 消灭所有地星 |
| 地星 | 消灭所有天星 |
| 散仙 | 存活且至少 2 名被你禁言过的玩家被放逐 |

## 🌟 角色一览

### 天星阵营
| 角色 | 技能 |
|------|------|
| 天宿星 | 斗转星移：受到伤害时转移给他人，首夜免疫 |
| 天权星 | 真权在握：投票后清零一名玩家票数 |
| 天遁星 | 天阵结印：每夜守护至多 2 名玩家免受伤害 |
| 天光星 | 天光显现：每夜可复活一名当夜死亡的玩家 |
| 天妄星 | 真视破妄：每夜查验一名玩家阵营 |
| 天衡星 | 天星永恒：每夜跟踪一名玩家，得知其是否使用技能 |
| 天斗星 | 仙陨星斗：散仙存活时双票权，散仙死后获得生死赌 |

### 地星阵营
| 角色 | 技能 |
|------|------|
| 地煞星 | 地灭天星：每夜击杀一名玩家 |
| 地爆星 | 地爆追星：被放逐时带走一名玩家 |
| 地毒星 | 地毒碎星：施毒，2 夜后毒发身亡 |
| 地阎星 | 阎罗禁咒：免疫地煞攻击并储存一次禁咒杀 |
| 地魅星 | 天生魅骨：死亡后标记一名玩家，其后续两次投票额外 +1 票 |
| 地影星 | 地影借仙：破除天遁守护并得知其身份 |

### 散仙
| 角色 | 技能 |
|------|------|
| 散仙 | 免疫击杀、首次被票可免死、每夜可禁言一名玩家 |

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 赛博朋克霓虹 UI 风格
- Font Awesome 图标库
- Google Fonts (Inter)

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可开始游戏。
